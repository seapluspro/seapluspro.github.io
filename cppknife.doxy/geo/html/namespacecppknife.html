Detailed Description

Implements an exception used for all classes of the scope geo.

Defines an abstract class for geometric items.

Implements a rectangle with vertical/horizonal lines.

Implements a point in a two dimension context.

Implements a translator from GPS to some EPSG/ESDI coorinates systems and vice versa.
