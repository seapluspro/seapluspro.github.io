Detailed Description

Manages a Comma Separated File (CSV).

A CSV file is a plain text file divided in rows and columns. Normally one row is one text line. The columns are separated by one character named "separator". To avoid ambiguities one column can be put into two delimiters: '"' or '\''. There are two ways to handle a CSV file:

  • traversing row by row
  • parsing the whole file at the begin
