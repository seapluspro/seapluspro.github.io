Detailed Description

Stores an amount of search patterns, with positive and negative patterns.

Stores an amount of search patterns, with positive and negative patterns.

Positive patterns: only texts matching that filter will be delivered. Negative pattterns: only text not matching the filter will be delivered.

Tests whether a string matches at least one of the patterns.

definition:
negative pattern: a pattern with the not condition
positive pattern: an other pattern 

Examples:
list: ";*.cpp;-test*;*.hpp"
returns true: "a.cpp" "xy.hpp"
returns false: "x.img", "test.cpp"
Parameters
toTestThe string to test.
toTestLength-1: strlen(toTest) will be used.
Otherwise: the length of toTest.
Returns
true: at least one pattern matches and no "not-pattern" matches
false: no pattern matches or at least one * "not-pattern" matches
