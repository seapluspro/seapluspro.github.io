◆ adapt() [1/2]

ChangeType cppknife::LineList::adapt ( const char *  pattern,
const char *  replacement,
const char *  anchor = nullptr,
bool  aboveAnchor = false 
)

Replaces a part of a line in the line list. The line to modify is the first line matching a given pattern. If the pattern is not found: the pattern anchor is searched. If the anchor is found the replacement is inserted depending on aboveAnchor. If the anchor is not found the replacement is inserted as last line. Example:

pattern: "max-memory="
replacement: "max-memory=512K"
anchor: "#.*max-memory"
Parameters
patternThe pattern of the line to modify as string.
replacementThe new value of the line after modifying.
anchorA pattern for finding the insertion position if pattern is not found.
aboveAnchortrue: the insertion position is above the anchor, otherwise below.
Returns
CT_UNCHANGED: the line matching pattern is exactly replacement. CT_CHANGED: the pattern has been found and the line is modified. CT_INSERTED: the pattern has not been found and the anchor has been found. CT_APPENDED: the pattern has not been found and the anchor has not been found.
