dbknife

Objectives

That program allows database administration for Mysql (Mariadb) and Postgres.

-u USER,--user=USER
Create a user. Format: <username>:<password>, e.g. --user=jonny:TopSecret
-a,--administrator
Sets the user rights as administrator., e.g. --administrator -a
-r,--read-only

OUTPUT
