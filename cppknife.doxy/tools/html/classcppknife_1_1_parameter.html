longnameThe name of the argument. If preceeded by "--" this implies an option and the name is that without "--".
shortnameThe short name of the argument. If preceeded by "-" this implies an option and the name is that without "-" only if name is nullptr.
dataTypeThe data type of the argument.
helpThe text used in the usage message.
defaultValueIf the parameter has no concrete program argument that value. is returned in asString(), asBool()...
examplesnullptr or a list of examples separated by '|', e.g. "0|-33|99"
multipletrue: The argument can occure multiple times.
